# 파이썬의 다양한 라이브러리를 사용하였습니다.

1) 머신러닝을 사용하여 사용자의 얼굴에 대해 흑백의 이미지로 캡쳐하여 학습하고

사용자의 얼굴에서 특징점을 검출하여 저장된 사용자와 비교하는 과정을 거쳐 유사율이 80%가 넘는다면 잠금을 해제하였습니다.
